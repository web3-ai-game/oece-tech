## 🏗️ 技术架构升级

### i18n系统
```typescript
// src/lib/i18n/
├── config.ts          # 语言配置
├── translations/
│   ├── en.json       # 英文
│   └── zh-TW.json    # 繁体中文
├── hooks.ts          # useTranslation hook
└── provider.tsx      # I18nProvider

// 使用方式
const { t, locale, setLocale } = useTranslation();
<h1>{t('visa.title')}</h1>
```

### Header/Footer组件
```typescript
// src/components/layout/

├── footer.tsx        # 品牌信息 + 链接
```
