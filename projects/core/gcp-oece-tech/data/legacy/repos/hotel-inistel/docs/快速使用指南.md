### 🌐 访问地址
- **React前端应用**: http://localhost:3000

### 🚀 常用命令

```bash
# 启动开发环境
./start.sh start

# 停止开发环境
./start.sh stop

# 重启服务
./start.sh restart

# 查看日志
./start.sh logs

# 清理环境（删除所有数据）
./start.sh clean
```

### 🐛 常见问题

**Q: 端口被占用怎么办？**
A: 停止服务后重启：`./start.sh stop && ./start.sh start`

**Q: 数据库连接失败？**
A: 检查端口5433是否被占用，或使用 `./start.sh logs` 查看错误信息

**Q: 如何重置数据？**
A: 使用 `./start.sh clean` 清理环境，然后 `./start.sh start` 重新开始

**Q: 如何添加新的数据库表？**
A: 修改 `database/init.sql` 文件，然后重新启动服务
