## 📐 尺寸優化

### 字體大小（移動端優先）
```css
/* 之前 */
text-3xl → 桌面30px, 移動太小

/* 現在 */
text-xl sm:text-2xl lg:text-3xl
移動18px → 平板24px → 桌面30px
```

### 卡片尺寸
```css
/* 之前 */
p-6 → 24px padding, 卡片過大

/* 現在 */
p-3 或 p-4 → 12-16px padding
更緊湊，信息密度提升
```

### 圓角半徑
```css
/* 之前 */
rounded-lg → 8px 圓角

/* 現在 */
rounded-2xl → 16px 圓角
更柔和，更現代
```
