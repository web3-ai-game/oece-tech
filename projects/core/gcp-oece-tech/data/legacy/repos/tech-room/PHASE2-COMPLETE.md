## 🚀 如何使用

### 1. 初始化數據庫

```bash
npm run db:init
```

### 2. 填充示例數據

```bash
npm run db:seed
```

### 3. 啟動開發服務器

```bash
npm run dev
```

### 4. 訪問教程頁面

- 教程列表: http://localhost:3000/tutorials
- 示例教程: http://localhost:3000/tutorials/solidity-smart-contracts-basics

## 📱 移動端測試

打開 Chrome DevTools:
1. 按 F12 打開開發者工具
2. 點擊 "Toggle device toolbar" (Ctrl+Shift+M)
3. 選擇不同設備測試:
   - iPhone SE (375px)
   - iPhone 12 Pro (390px)
   - iPad (768px)
   - iPad Pro (1024px)

## 🎯 核心功能演示

### 搜索和篩選
1. 在搜索框輸入關鍵詞（如 "Solidity"）
2. 選擇分類（如 "Web3 開發"）
3. 選擇難度（如 "入門"）
4. 實時查看篩選結果

### 查看教程
1. 點擊任意教程卡片
2. 閱讀 Markdown 格式的內容
3. 查看代碼示例（帶語法高亮）
4. 點擊"複製代碼"按鈕
5. 點贊或收藏教程

### 分享功能
- 點擊"分享"按鈕
- 在支持 Web Share API 的設備上（如手機），會彈出原生分享菜單
- 可分享到微信、WhatsApp 等

## 💡 技術亮點

### Markdown 渲染
使用 `react-markdown` 和自定義組件，完美支持：
- 標題（H1-H6）
- 段落和列表
- 代碼塊和行內代碼
- 引用

- 強調和斜體

### 代碼高亮

- 複製功能

- Tailwind 的響應式工具類
- Mobile-first 方法
- 觸摸優化的按鈕
- 自適應文字大小
