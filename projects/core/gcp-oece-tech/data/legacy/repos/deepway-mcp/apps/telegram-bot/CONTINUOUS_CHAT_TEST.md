# 🎯 连续对话测试指南

## ✅ 新功能：触发后连续接话5次

### 工作原理

1. **触发**: 用户说 "小爱 你好"
2. **第1轮**: 小爱回复 `@用户 xxx [1/5]`
3. **第2轮**: 用户说任何话（不需要再说"小爱"）
4. **第2轮**: 小爱自动回复 `@用户 xxx [2/5]`
5. **持续5轮**
6. **5轮后**: 自动结束，需要重新触发

---

## 🧪 完整测试场景

### 场景1: 连续对话

```
用户: 小爱 你好
小爱: @用户 我明白你的感受 😊 [1/5]

用户: 今天天气不错
小爱: @用户 让我来帮你 💝 [2/5]

用户: 你觉得呢
小爱: @用户 我一直在这里 🌟 [3/5]

用户: 谢谢你
小爱: @用户 别担心，慢慢说 💫 [4/5]

用户: 再见
小爱: @用户 我懂你的意思 ✨ [5/5]

用户: 还在吗？
小爱: (不回复，需要重新触发)
```

### 场景2: 重新触发

```
用户: 小爱 帮忙
小爱: @用户 我明白你的感受 😊 [1/5]

(开始新的5轮对话)
```

### 场景3: 多用户独立

```
用户A: 小爱 你好
小爱: @用户A xxx [1/5]

用户B: 小爱 帮忙
小爱: @用户B xxx [1/5]

用户A: 继续聊
小爱: @用户A xxx [2/5]

用户B: 我也继续
小爱: @用户B xxx [2/5]
```

## 📊 状态查询

### 查看当前轮次
```bash
# 查看所有对话
redis-cli keys "group_round:*"

# 查看特定用户
redis-cli get "group_round:xiaoai:-1001234567890:123456789"
```

### 清空轮次（重新开始）
```bash
redis-cli del "group_round:xiaoai:-1001234567890:123456789"
```
