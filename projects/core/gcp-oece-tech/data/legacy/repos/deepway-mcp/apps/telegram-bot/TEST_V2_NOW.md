## 📊 查看轮次

```bash

redis-cli keys "conv:*"

# 查看特定用户轮次
redis-cli get "conv:{chat_id}:{user_id}"

# 示例
redis-cli get "conv:-1001234567890:123456789"
```

## 🐛 如果没有反应

### 1. 确认Bot运行
```bash
ps aux | grep multi_bot_v2
```

### 2. 确认群组隐私模式关闭
```
@BotFather → /mybots → @svskilo_bot
→ Bot Settings → Group Privacy → Turn Off
```

### 3. 重启Bot
```bash
pkill -f multi_bot_v2
./run_multi_bots_v2.sh
```
