# Monthly Pool Budget Template（不含 Gemini）

> 说明：Gemini API 按日重置，不参与本模板月度预算。此模板用于对“按月计费/配额”的服务进行月→日安全配额换算，并指导调度与降频策略。

## 计算规则
- 每日安全限额 = ⌊ 月度配额 × 安全系数(0.8) ÷ 30 ⌋
- 峰值日（活动/高峰）可临时放宽至 1.2×，随后以 0.6× 回补。
- 当日>80% 预警，>90% 进入降频或停止可选项。

## 模板
| 服务 | 月度配额/上限 | 安全系数 | 每日安全限额 | 峰值日上限 | 降频策略 | 备注 |
|---|---:|---:|---:|---:|---|---|
| Datadog | N 次指标写入 | 0.8 | ⌊N×0.8/30⌋ | 日预算×1.2 | 降低采样/关闭部分自定义指标 | 免费层优先 |
| Supabase/Postgres | M 次写入/计算 | 0.8 | ⌊M×0.8/30⌋ | 日预算×1.2 | 拉长 Worker 间隔（3h→6h→12h） | 或自建 Postgres |
| Deepnote/Camber | K 次运行 | 0.8 | ⌊K×0.8/30⌋ | 日预算×1.2 | 合并报表/周报优先 | 用作分析非主库 |
| Notion | R 次 API 调用 | 0.8 | ⌊R×0.8/30⌋ | 日预算×1.2 | 只在必要时同步 | |
| GitHub | CI 分钟/调用 | 0.8 | ⌊配额×0.8/30⌋ | 日预算×1.2 | 合并工作流/夜间跑 | |
| OpenRouter | $B 月预算 | 0.8 | ⌊($B×0.8)/30⌋ | 日预算×1.2 | 限流+模型替代 | 非主路由 |
| DigitalOcean | 流量/对象存储 | 0.8 | ⌊配额×0.8/30⌋ | 日预算×1.2 | 开启缓存/CDN | |

## 执行建议
- 归档 Worker：默认 3 小时；预算紧张时扩至 6~12 小时。
- 报表：按周固定输出；峰值周额外一次。
- 降频：达 80% 时发预警；达 90% 自动减载（停选配、延长间隔）。

## 记录区（示例）
- 2025-11：
  - Datadog：月 30k → 日 800（安全）；峰值日 960；当周周报 1 次
  - Supabase：月 90k 写入 → 日 2,400（安全）；峰值日 2,880；Worker 6 小时
