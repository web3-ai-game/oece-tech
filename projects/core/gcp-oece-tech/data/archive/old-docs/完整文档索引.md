## 📋 项目概览

### 两个网站系统的定义

我们正在将分散的 27 个 GitHub 项目重组为两个独立的、功能完整的网站生态系统：

#### 🌐 **oece.tech** - 极客母艦生态
**定位**: 综合导航和知识中心  
**核心功能**:
- 📊 项目导航和展示
- 🔍 全文搜索 (Algolia)
- 📚 知识库集成
- 🤖 AI 工具展示
- 💬 Telegram Bot

**包含项目** (7 个):
- oece-tech (核心)
- web3-ai-game
- sms-agentic-tg-bot
- 420420
- db + legacy-repos-archive
- gcp-dev-environment
- gcp-sms-deployment

**部署**: Firebase Hosting + GCP  
**成本驱动**: Algolia 搜索费用

---

#### 🌐 **deepweay.me** - 深维世界
**定位**: AI 聊天和内容生成平台  
**核心功能**:
- 🧠 Gemini AI 深度对话
- 💾 聊天历史和管理
- 📚 知识库集成 (小说库)
- ✨ 内容生成和优化
- 📱 电子书转换工具
- 🤖 Telegram Love Bot

**包含项目** (8 个):
- deepweay-me (核心)
- deepweay-digital-gold-vault
- ebook-converter
- notes
- MD (1016 本小说)
- deepseet-knowledge-base
- gcp-distilled-knowledge
- do-n8n

**部署**: Firebase Hosting + GCP  
**成本驱动**: Gemini API 调用费用

---

### 系统架构概览

```
Web3 生态 (web3-ai-game)
│
├── 🌐 oece.tech (极客生态)
│   ├── 前端: Next.js 16 + React 19
│   ├── 数据库: MongoDB (oece_tech_db)
│   ├── 搜索: Algolia
│   ├── 监控: Sentry
│   ├── 部署: Firebase + GCP
│   └── 成本: 主要为搜索费用
│
└── 🌐 deepweay.me (深维世界)
    ├── 前端: Next.js 16 + React 19
    ├── 数据库: Firebase Realtime DB + MongoDB
    ├── AI: Google Gemini Pro API
    ├── 监控: Sentry
    ├── 部署: Firebase + GCP
    └── 成本: 主要为 AI API 调用
```
