“这位是薛二小姐，湘灵。”宁王看向女子，介绍道。

她看着眼前这个笑靥如花的女子，心中却涌起一丝莫名的距离感。

“王爷，这位姑娘是谁？”

一个清脆的声音响起，薛湘灵回过神来，只见一个年轻的女子走了过来，她一身粉色衣裙，头上戴着精致的步摇，眉眼间带着几分娇俏。

## 第八章：暗潮涌动

“小姐，您可要多加小心，大小姐说，王爷对您，恐怕是别有用心的。”

薛湘灵点了点头，心中暗自警惕。

她穿越到这里，本就身不由己，更不想卷入什么权力斗争。

“小桃，你说，这平国公府里，到底是谁在掌权？”

薛湘灵心中一惊，宁王妃，慕容瑾，她终于来了。

薛湘灵连忙上前行礼。“王妃姐姐，你好。”

她看着两位王爷，心中暗自思忖，这宁王府，只怕比她想象的还要复杂。

## 第九章：家宴

“王爷。多日不见，本王甚是想念。”宁王说着，便走到了薛湘灵身边，亲手为她夹了一块菜。

薛湘灵心中更加警惕，这宁王，对自己的情意，只怕已经昭然若揭。
