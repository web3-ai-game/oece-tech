## 1821
> 2007-10-03 11:15

29岁的医生，医院的骨干，医院的教授。他最近一直在思考如何才能成为一个更好的医生。

“我需要更多的知识，更多的经验，”他对自己的想法是，“我必须不断学习，不断进步，才能得到患者的信任。”

他是一位非常成功的医生，深受患者的喜爱。但是，他知道自己还有很多不足之处。

“我必须成为一个优秀的医生，才能为患者带来更多的健康。”

他开始更加努力地工作，学习更多的知识，参加更多的会议，学习更多的技术。

他是一位非常杰出的医生，深受患者的喜爱。

有一天，一位患者对他说：“医生，您真是太棒了！您不仅医术精湛，而且医德高尚。”

听到患者的称赞，他感到非常欣慰。

“我是一名医生，这是我的职责。”

他继续努力工作，学习更多的知识，学习更多的技术，为患者带来更多的健康。
