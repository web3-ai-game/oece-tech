# 北之星冠

## 1. 概述

北之星冠（约公元1073年）是来自维吾尔语的《开善疏勒经》的另一段经文，它是《般若波罗蜜多心经》的注释，也是《北之星冠》的缩影。

> 《北之星冠》可能是中国唐朝的佛教著作，其作者为敦煌的维吾尔学者。

《北之星冠》的作者是维吾尔学者，大约生活在中国唐朝时期，是来自敦煌的学者。

## 2. 《北之星冠》的发现

《北之星冠》最初以维吾尔语出版，在1073年被发现。

## 3. 《北之星冠》的版本

《北之星冠》的第一个版本出版于1075年。
