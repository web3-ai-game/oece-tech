# 《一颗水蜜桃》

## chapter1

在一个喧闹的夜晚，城市霓虹闪烁，但这一切都无法吸引林薇的注意力。她坐在窗边，望着窗外漆黑的夜空，手中紧握着一张泛黄的旧照片。照片上，一个年轻的男人微笑着，眼神中充满了希望。那是她的初恋，顾北辰。

“北辰，你现在在哪里？”她喃喃自语，声音中带着一丝哽咽。

林薇从小就生活在一个破碎的家庭，父母常年争吵，母亲在她很小的时候就离开了，只留下她和沉默寡言的父亲。顾北辰的出现，像一缕阳光照进了她灰暗的生活。他阳光开朗，幽默风趣，给了她从未有过的温暖和安全感。

然而，命运似乎总爱开玩笑。就在他们即将迎来美好未来的时候，顾北辰因为一场突如其来的事故，永远地离开了她。

“哥，你说，我要不要告诉爸关于北辰的事？”林薇的弟弟林宇小心翼翼地问。

林宇知道姐姐一直很痛苦，顾北辰的离开对她的打击太大了。他担心姐姐会因此沉沦下去，但同时，他也知道哥哥林峰不喜欢顾北辰。

“别，现在不是时候。”林宇说，“爸现在心情不好，等他情绪稳定了再说。”

林宇走后，林薇看着镜子里的自己，眼眶红红的，但她努力挤出笑容。她知道，不能一直沉浸在悲伤中，她还有父亲和弟弟需要她。

“北辰，我会坚强下去，等你回来。”她对着照片，轻声说道。

## chapter2

大学生活就像一场盛大的烟花，绚烂而短暂。林薇在校园里遇到了很多人，也经历了很多事。她努力学习，积极参加社团活动，试图将自己的生活填满，好让顾北辰的影子不再那么清晰。

“林薇，下午有个社团活动，你要不要来？”室友王晓晓兴奋地邀请道。

王晓晓是林薇最好的朋友，她善良活泼，总是能带给林薇快乐。

“好啊，我去。”林薇笑着答应。

在活动中，林薇遇到了一个叫陆子铭的男生。他温文尔雅，才华横溢，身上散发着一种成熟男人的魅力。他的出现，像一阵清风，慢慢吹散了林薇心中的阴霾。

“林薇，你很特别。”陆子铭看着林薇，眼神中带着一丝赞赏。

林薇有些意外，她觉得自己并没有什么特别之处。

“我只是一个普通的学生。”她微笑着说。

“不，你不是。”陆子铭说，“你的眼神中有一种别人没有的坚韧，那是经历过风雨才能有的。”

林薇的心被触动了，她看着陆子铭，第一次感觉到，自己也许可以重新开始。

## chapter3

毕业后，林薇进入了一家不错的公司工作，陆子铭也成为她的同事。他们的关系在工作中不断升温，很快就走到了一起。陆子铭对林薇非常好，他体贴入微，关心备至，让林薇感受到了前所未有的幸福。

“子铭，谢谢你。”林薇看着陆子铭，眼神中充满了感激。

“傻瓜，我们是夫妻，说什么谢。”陆子铭温柔地抚摸着林薇的头发，“我只想让你幸福。”

林薇笑了，她知道自己找到了对的人。

然而，平静的生活并没有持续多久。有一天，林薇接到了一个陌生电话。

“请问是林薇女士吗？”电话那头传来一个男人的声音，“我是顾北辰的朋友，有件事想告诉你。”

林薇的心猛地一沉，顾北辰的名字，像一把钝刀，在她心里狠狠地搅动了一下。

“你说。”她声音有些颤抖。

“顾北辰并没有死。”男人说，“他当年是因为一场意外，被送往国外治疗，现在他已经回来了，并且……”

电话那头顿了一下，似乎在斟酌词句。

“并且，他现在是这家公司的老板。”

林薇的大脑一片空白，她无法相信自己听到的。顾北辰？回来了？而且是这家公司的老板？

## chapter4

林薇坐在办公桌前，脑海里全是那个男人微笑的脸。她无法集中精神，手里不停地转着笔。

“怎么了？一脸心事重重的。”陆子铭走过来，关切地问道。

“没什么，只是有点累。”林薇勉强挤出笑容。

陆子铭看着林薇，眼神中带着一丝担忧。他知道，林薇心里藏着一个秘密，一个关于顾北辰的秘密。

“林薇，你是不是有什么事情瞒着我？”陆子铭轻声问道。

林薇看着陆子铭，心里一阵愧疚。她不知道该如何开口，告诉他关于顾北辰的事情。

“子铭，我……”她欲言又止。

“没关系，你说。”陆子铭温柔地说，“无论发生什么，我都会在你身边。”

林薇深吸一口气，决定把一切都告诉陆子铭。

## chapter5

林薇将顾北辰的事情告诉了陆子铭。陆子铭听完后，并没有生气，反而更加心疼林薇。

“林薇，我理解你的感受。”陆子铭紧紧地抱住林薇，“我知道顾北辰对你很重要。但是，现在你身边有我，我会好好照顾你，让你幸福。”

林薇感动得落泪，她知道，自己没有看错陆子铭。

“谢谢你，子铭。”她哽咽着说。

“别谢，我们是夫妻。”陆子铭温柔地说，“你只需要记住，我永远在这里。”

林薇靠在陆子铭的怀里，感受着他的温暖。她知道，自己已经找到了幸福，也该放下过去，好好珍惜眼前人了。

## chapter6

林薇和陆子铭的生活又恢复了平静。然而，命运似乎并不想放过他们。

有一天，林薇在公司遇到了顾北辰。他依然那么阳光帅气，眼神中依然充满了那种熟悉的温柔。

“林薇？”顾北辰看着林薇，眼神中充满了惊讶。

林薇看着顾北辰，心中百感交集。

“北辰？”她声音有些颤抖。

“是你？真的是你？”顾北辰激动地上前，紧紧地抱住林薇。

林薇被顾北辰的拥抱弄得有些不知所措。她已经有了陆子铭，她不知道该如何面对顾北辰。

“北辰，我已经结婚了。”她轻声说道。

顾北辰的身体僵了一下，他看着林薇，眼神中充满了难以置信。

“结婚了？和谁？”他问道。

“陆子铭。”林薇说道。

顾北辰的脸色瞬间变得苍白，他看着林薇，眼神中充满了失落。

“原来是这样。”他喃喃自语，声音中带着一丝痛苦。

## chapter7

顾北辰的出现，打破了林薇平静的生活。她夹在两个男人之间，进退两难。

“林薇，你到底爱谁？”陆子铭看着林薇，眼神中充满了痛苦。

“子铭，我……”林薇无法回答。

“我再问你一遍，你到底爱谁？”陆子铭的声音变得有些激动。

林薇看着陆子铭，心里一阵愧疚。她知道，自己不能再犹豫了。

“子铭，对不起。”她哽咽着说，“我爱的是你。”

陆子铭看着林薇，眼神中充满了惊喜。

“真的吗？”他激动地问道。

“嗯。”林薇点头。

陆子铭紧紧地抱住林薇，他知道，自己终于赢得了她的心。

## chapter8

顾北辰得知林薇选择了陆子铭后，伤心欲绝。他无法接受这个事实，他觉得自己失去了生命中最重要的一切。

“为什么？为什么你不选择我？”他痛苦地嘶吼着。

顾北辰走在城市的街头，他觉得自己仿佛置身于一个黑暗的迷宫，找不到出路。

“林薇，我爱你，我真的爱你。”他对着夜空，痛苦地说道。
