## 游戏化术语词典

### A - 装备与工具

```
装备 (Equipment) = 软件/工具
- 传送装备 = VPN客户端
- 中继装置 = 代理工具
- 探索工具包 = 浏览器插件
- 隐身斗篷 = 匿名工具
- 加速靴 = 网络加速器
```

### B - 地点与位置

```
区域 (Region) = 网络空间
- 本土区域 = 国内网络
- 跨界区域 = 需要特殊访问
- 全球地图 = 国际互联网
- 隐藏副本 = 特殊网站
- 禁区 = 被封锁区域
```

### C - 行动与操作

```
行动 (Action) = 网络操作
- 开启传送 = 连接VPN
- 切换据点 = 更换服务器
- 激活中继 = 启用代理
- 探索地图 = 浏览网站
- 建立基地 = 搭建服务器
```

### D - 状态与效果

```
状态 (Status) = 连接状态
- 传送中 = 连接中
- 已到达 = 连接成功
- 传送失败 = 连接失败
- 隐身状态 = 匿名模式
- 全速前进 = 无限制访问
```
