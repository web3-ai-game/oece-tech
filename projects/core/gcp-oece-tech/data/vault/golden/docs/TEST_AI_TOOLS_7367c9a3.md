## ⚠️ 可能遇到的问题

### 问题1: API Key错误
```
Error: GEMINI_API_KEY is not set
```
**解决：** 检查 `.env.local` 文件是否有 `GEMINI_API_KEY`

### 问题2: JSON解析错误
```
Failed to parse JSON response
```
**原因：** AI返回格式不标准
**解决：** 重试一次，或查看console日志

### 问题3: 响应太慢
```
等待超过20秒
```
**原因：** Gemini API限流或网络问题
**解决：** 等待或稍后重试

### 问题4: i18n错误
```
t.visaToolTitle is undefined
```
**解决：** 检查i18n配置，或直接用英文文本
