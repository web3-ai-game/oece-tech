## 🔐 匿名机制详解

### 前台显示
```
用户A发帖 → 显示为"神秘駭客#1337"
用户B看到 → "神秘駭客#1337"（匿名）
用户A自己看 → "我的用户名 (你)"（真实+标识）
```

### 后台记录
```sql
-- 数据库存储真实ID（管理员可见）
author_id: 123 (真实)
display_mode: 'anonymous'
anonymous_name: '神秘駭客#1337'

-- IP追踪（防滥用）
ip_address: '192.168.1.1'
user_agent: 'Mozilla/5.0...'
```

### 安全保障
- ✅ API不返回真实用户信息
- ✅ IP追踪在后台（用于举报）
- ✅ 举报机制
- ✅ 管理员可查看真实身份（仅在违规时）
