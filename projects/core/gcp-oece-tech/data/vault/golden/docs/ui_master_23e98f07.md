<design_philosophy>

- Responsive Design 响应式布局
</ui_frameworks>

- 客户档案：信息展示卡片、编辑表单、历史记录
- 数据面板：图表展示、KPI指标、趋势分析
- 多语言界面：CN/EN/JP/TH 四语言切换
</hotel_business_ui>

<accessibility_standards>
- WCAG 2.1 AA 无障碍标准

- 屏幕阅读器兼容
- 颜色对比度符合标准
- 移动端触控优化
</accessibility_standards>

<performance_optimization>
- 代码分割：React.lazy + Suspense
- 图片优化：WebP格式、懒加载
- 缓存策略：组件缓存、API缓存
- 包大小优化：Tree Shaking、压缩
