# **App Name**: Digital Nomad Compass

## Core Features:

- Article Browsing: Browse the latest articles through a card-based interface.
- Category Filtering: Filter articles by 'Tech' or 'Travel' categories.
- Comprehensive Search: Search for specific topics using a comprehensive search tool.
- Detailed Article View: Immersive reading experience with detailed, multimedia-rich articles.
- AI Trip Planner: AI tool that creates personalized travel itineraries based on user preferences, integrating tech insights relevant to the region.

## Style Guidelines:

- Primary color: Warm amber (#FFB300) to evoke a sense of adventure and warmth.
- Background color: Deep charcoal gray (#333333) for a modern, dark theme.
- Accent color: Soft cyan (#77D8D8) as a contrasting accent to provide highlights and guide user actions.
- Body font: 'Inter', a sans-serif font, providing a modern and readable experience.
- Headline font: 'Space Grotesk', a sans-serif font, for a computerized, techy feel.
- Use minimalistic icons that represent both technology and travel, with a style consistent with the dark theme.
- Card-based layout for article previews, optimized for responsiveness on different devices.
- Subtle transitions and animations to enhance user experience when navigating between articles and categories.
