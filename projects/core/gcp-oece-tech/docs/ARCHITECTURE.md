# oece.tech 架构设计文档

**最后更新**: 2026-01-15

## 系统概览

**oece.tech** 是一个基于 Next.js 16 的极客项目导航平台，提供:
- 项目展示和管理
- Algolia 全文搜索
- 知识库集成
- Telegram Bot 集成

## 数据流

1. **用户请求** → Next.js SSR
2. **搜索查询** → Algolia → 返回结果
3. **数据读取** → MongoDB → 缓存到 Redis
4. **Telegram Bot** → Webhook → API Routes

## 部署架构

- **前端**: Firebase Hosting + CDN
- **API**: Firebase Cloud Functions
- **数据库**: MongoDB Atlas
- **搜索**: Algolia Cloud
- **监控**: Sentry + GCP Logging
