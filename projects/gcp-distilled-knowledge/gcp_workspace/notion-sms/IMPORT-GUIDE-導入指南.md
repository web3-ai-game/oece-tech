# 📥 Notion 導入指南

## 🚀 快速導入 (3 步驟)

### Step 1: 解壓文件
```bash
unzip notion-sms-purified-v2.zip
```

### Step 2: 打開 Notion
1. 進入你的 Notion 工作空間
2. 點擊左側邊欄的 **Settings & Members**
3. 選擇 **Import** 標籤頁

### Step 3: 導入文件
1. 點擊 **Import** 按鈕
2. 選擇 **Markdown & CSV**
3. 選擇解壓後的 **整個文件夾** 或 **單個 .md 文件**
4. 確認導入

## 📁 目錄結構

導入後會在 Notion 中創建以下結構:

```
📡 Notion-SMS (根頁面)
│
├── 🧠 01-AI戰略核心
├── 🚀 02-DeepWeay產品矩陣
├── 🌍 03-賽博宇宙觀
├── ⚙️ 04-OECE工程體系
├── 📚 05-資源與工具
└── 🔒 06-敏感數據-限制訪問 (建議刪除或加密)
```

## ⚠️ 重要提示

### 敏感數據處理

**導入前強烈建議**:
1. 刪除 `06-敏感數據-限制訪問/` 整個目錄
2. 或在導入後立即設置該頁面為私密
3. 確保不會分享到公開鏈接

該目錄包含:
- 🔴 Slack 2FA 備份碼
- ⚠️ API Keys & Tokens
- ⚠️ 服務器配置信息

### Notion 格式支持

✅ **支持的功能**:
- Markdown 標題 (H1-H6)
- 列表 (有序/無序)
- 代碼塊 (含語法高亮)
- 表格
- 鏈接和圖片
- 引用塊

⚠️ **可能需要調整**:
- 某些 Emoji 可能顯示不同
- 複雜表格可能需要手動調整
- 自定義 Markdown 擴展可能不支持

## 🎯 導入後建議

### 1. 設置權限
- 檢查所有頁面的分享權限
- 敏感頁面設為 "私密"
- 團隊頁面設為 "工作空間可見"

### 2. 添加標籤
建議為每個頁面添加標籤:
- #AI戰略
- #產品開發
- #技術架構
- #資源庫

### 3. 創建關聯
使用 Notion 的 `@提及` 功能:
- 在相關頁面間創建雙向鏈接
- 構建知識圖譜網絡

### 4. 自定義樣式
- 添加封面圖片
- 設置圖標
- 調整字體大小
- 配置顏色主題

## 🔧 疑難排解

### Q: 導入後格式混亂?
A: Notion 會自動轉換 Markdown,某些複雜格式需要手動調整。

### Q: 中文顯示亂碼?
A: 確保文件編碼為 UTF-8,重新導入。

### Q: 文件夾結構沒保留?
A: 導入時選擇 "保留目錄結構" 選項。

### Q: 代碼塊沒有高亮?
A: 在 Notion 中手動設置代碼塊語言。
