# Working Copy：iOS Git 客戶端完整指南

## 介紹
Working Copy 是一款功能完整的 iOS Git 客戶端，專為 iPhone 和 iPad 設計，讓使用者在行動裝置上無縫管理 Git 儲存庫。它支援完整的 Git 工作流程，包括克隆、編輯、提交和協作功能，特別適合開發者和學生使用。透過 GitHub Student Developer Pack，學生可免費解鎖所有 Pro 功能（正常售價 $35.99 USD）。

## 主要功能
Working Copy 提供專業級 Git 體驗，讓您在 iOS 裝置上輕鬆處理專案：

- **克隆儲存庫**：直接從 GitHub 或其他 Git 託管平台克隆儲存庫到您的 iPhone 或 iPad。
- **編輯與版本控制**：在內建編輯器中修改檔案，進行提交 (commit) 和推送 (push) 到遠端儲存庫。
- **拉取請求 (Pull Requests)**：開啟和參與 PR，提升團隊協作效率。
- **跨 App 存取**：儲存庫檔案可在裝置上的其他應用程式（如文字編輯器或 IDE）中開啟和編輯，實現無縫整合。
- **Pro 功能**：包含進階工具如分支管理、合併衝突解析等（學生免費解鎖）。

這些功能讓 Working Copy 成為行動 Git 管理的首選，無需依賴電腦即可完成完整開發循環。

## 安裝與更新
1. 開啟 **App Store**。
2. 搜尋 **Working Copy**。
3. 安裝或更新至最新版本（適用 iPhone 和 iPad）。

安裝後，即可立即開始使用基本功能。
