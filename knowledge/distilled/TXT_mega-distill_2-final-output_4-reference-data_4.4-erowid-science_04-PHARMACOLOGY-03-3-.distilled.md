# 藥物藥代動力學與藥物相互作用知識文檔

## 引言
本知識文檔基於提供的關鍵事實，詳細探討藥物藥代動力學（Pharmacokinetics, PK）與藥物相互作用（Drug Interactions）的核心概念。藥代動力學研究藥物在體內的**吸收（Absorption）、分佈（Distribution）、代謝（Metabolism）和排泄（Excretion）**，簡稱ADME過程。這些因素直接影響藥效、安全性和劑量設計。藥物相互作用則探討多種物質組合時的協同或拮抗效應，常見於娛樂性藥物或醫療情境中。本文補充科學脈絡、臨床背景，並保持事實準確。**注意**：此為教育性文件，非醫療建議；藥物使用涉及風險，請諮詢專業人士。

## 1. 藥代動力學 (Pharmacokinetics)

### 1.1 給藥途徑比較
給藥途徑決定藥物的**吸收速度**、**生物利用度（Bioavailability, F）**和**起效時間**。生物利用度指進入系統循環的有效藥物比例（0-100%），受途徑、藥物性質（如脂溶性）和第一過效應（肝臟首過代謝）影響。注射途徑生物利用度最高（近100%），口服最低（常<50%）。

以下表格比較常見途徑，包含優缺點及示例物質（多為精神活性藥物，用於說明吸收差異）：

| 途徑     | 優點                  | 缺點              | 示例物質    | 脈絡說明 |
|----------|-----------------------|-------------------|-------------|----------|
| **吸入** | 快速吸收（肺泡豐富血管，起效<1分鐘） | 肺損傷風險高     | DMT        | 用於氣溶膠或蒸氣，適合短效幻覺劑。 |
| **鼻內** | 方便、快速（鼻黏膜血管豐富） | 鼻膜刺激、損傷   | Cocaine    | 粉末形式，常見於娛樂使用，但長期損鼻中隔。 |
| **舌下** | 避開肝臟首過效應，直接入血 | 味苦、需維持接觸 | LSD        | 薄膜或液體，生物利用度高，避免胃酸降解。 |
| **口服** | 易用、非侵入性       | 第一過效應（肝臟代謝>50%） | Psilocybin | 膠囊/錠劑，起效慢（30-90分鐘），適合持續效應。 |
| **注射** | 精準劑量、100%生物利用度、緊急用 | 感染/HIV風險、靜脈損傷 | Heroin     | 醫療（如腎上腺素）或濫用，起效即刻。 |

**脈絡補充**：
- **吸收速度排序**：注射 > 吸入 > 鼻內/舌下 > 口服。
- **臨床應用**：注射用於緊急醫療（如心臟驟停）；口服適合慢性治療。
- **風險**：非醫學途徑增加過量或污染風險（如靜脈注射細菌感染）。

### 1.2 生物利用度計算範例
生物利用度可用簡單模型估計。以下Python代碼示範基本計算（假設靜脈注射為100%基準）：

```python
def bioavailability(route='Oral'):
    """估計生物利用度（百分比，簡化模型）"""
    rates = {
        'Injection': 100,   # IV/IM，無首過效應
        'Inhalation': 90,   # 高但有肺損失
        'Intranasal': 70,   # 鼻黏膜吸收
        'Sublingual': 80,   # 避肝臟
        'Oral': 50          # 典型首過效應
    }
    return rates.get(route, 0)

# 示例輸出
print(bioavailability('Injection'))   # 100
print(bioavailability('Oral'))        # 50
print(bioavailability('Inhalation'))  # 90
```

**解釋**：此模型基於文獻平均值（如口服psychedelics F≈40-60%）。實際值依個體（體重、酶活性）變異，可用AUC（曲線下面積）精算：F = (AUC_oral / AUC_IV) × 100%。

## 2. 藥物相互作用 (Drug Interactions)

### 2.1 危險組合 (3.21 危險組合)
**背景**：全球opioid危機（美國CDC數據：2022年>10萬過量死亡）多因組合使用，如opioids + 鎮靜劑。娛樂藥物（如benzodiazepines + opioids）放大風險。

**原理**：
- **協同效應（Synergism）**：藥物共同增強效應，如呼吸抑制（opioids壓低呼吸中樞，酒精加劇）。
- **酶抑制**：一藥抑制CYP450酶，延長另一藥半衰期（e.g., Grapefruit汁阻CYP3A4）。
- **示例**：Alcohol + Opioids → 呼吸停止；Stimulants (Cocaine) + MAOIs → 高血壓危機。

**減害建議**：避免組合；監測劑量；使用測試劑（e.g., fentanyl試紙）。

### 2.2 減害組合 (3.22 減害組合)
**實例**：**Naloxone（納洛酮）**，μ-opioid受體拮抗劑，反轉opioid過量。靜脈/鼻噴拯救無數生命（WHO：每年數十萬例）。
- **機制**：競爭性阻斷，恢復呼吸（效期30-90分鐘）。
- **脈絡**：美國廣泛分發“雙劑量套件”；與buprenorphine組合用於戒癮。
- **其他減害**：Vit C緩衝注射溶液（減血管損傷）；Magnesium補充電解質失衡。

**數據支持**：Naloxone自2010年起，美國過量死亡率下降20-50%（CDC）。

## 結論與注意事項
藥代動力學決定“藥物如何影響身體”，相互作用決定“多藥如何互動”。選擇途徑時，權衡速度 vs. 風險；避免危險組合，優先減害工具如Naloxone。**免責**：此文教育用途，非鼓勵使用。藥物濫用致死風險高；求助專業醫療或熱線（如SAMHSA：1-800-662-HELP）。參考來源：FDA、PubMed、Erowid（減害數據）。
