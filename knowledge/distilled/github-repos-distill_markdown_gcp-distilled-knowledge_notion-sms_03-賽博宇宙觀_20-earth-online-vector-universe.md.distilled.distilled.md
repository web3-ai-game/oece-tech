# 地球Online·记忆向量宇宙（Earth Online Vector Universe）知識文檔

## 項目概述

**地球Online·记忆向量宇宙**（簡稱 EOVU）是一個創新的 AI 驅動賽博社會實驗平台，旨在探索人類行為的數字化表達與量化。該項目將用戶互動轉化為高維數據結構，模擬「數字靈魂」的概念，結合賽博朋克美學與先進 AI 技術，構建一個虛擬社會實驗場域。EOVU 的靈感源自於神經科學、AI 行為建模與元宇宙概念，目標是透過大規模用戶參與，揭示人性模式並生成可驗證的「靈魂座標」。

項目定位為 Web3 時代的「行為經濟學實驗室」，用戶不僅是參與者，更是數據主權持有者。透過 Gemini AI 的強大自然語言處理能力，平台將用戶的對話、決策與情感輸入編碼為「記憶向量」（高維數學表示），形成獨一無二的數字身份證明。這不僅是娛樂工具，更是對人類意識數字化的前瞻性嘗試。

## 核心目標

EOVU 的使命是透過系統化實驗，量化並預測人類行為模式。具體目標包括：

- **AI 驅動的賽博社會實驗平台**：模擬真實社會互動，讓用戶在虛擬環境中進行角色扮演、決策遊戲與社交實驗，收集高質量行為數據。
- **通過人性實驗量化用戶行為**：設計多輪互動挑戰（如道德困境、合作博弈），分析用戶回應，生成量化指標（如信任分數、風險偏好）。
- **利用 Gemini AI 將用戶行為編碼為高維“記憶向量”（灵魂坐标）**：Gemini AI 負責將原始輸入轉化為 1024+ 維向量空間表示，每個向量捕捉用戶的「人性光譜」（情感、認知、道德維度），類似於嵌入式表示（embedding），但專注於長期記憶與人格建模。
- **構建數字靈魂證明**：最終輸出「數字靈魂 NFT」或可驗證憑證，證明用戶的行為模式唯一性，可用於元宇宙身份驗證、招聘匹配或娛樂應用。

這些目標支撐了平台的長期願景：建立全球最大的人性數據庫，推動 AI 倫理與行為科學的進步。

## 用戶功能

平台為用戶提供沉浸式體驗，強調個性化與隱私（數據僅用戶擁有）。核心功能如下：

| 功能 | 描述 | 解鎖方式 |
|------|------|----------|
| **個性化賽博身份檔案** | 基於記憶向量生成的完整數字肖像，包括視覺化「靈魂地圖」。 | 付費解鎖（基礎/進階/終極版） |
| **完整向量數據** | 高維記憶向量下載（JSON/可視化圖譜），支援第三方 AI 分析。 | 付費後即時生成 |
| **人性光譜分析** | 360° 人格評估（如 Big Five + 自訂維度：野心、共情、適應力）。 | 包含在檔案中 |
| **命運預測模擬** | AI 基於向量預測用戶在虛擬情景下的「命運軌跡」（職業、關係、危機應對）。 | 進階付費功能 |
| **免費試玩** | 基礎實驗參與，生成簡易向量預覽。 | 註冊即用 |

用戶可透過 Web/App 參與實驗，數據實時更新，確保動態演化。

## 商業模式

EOVU 採用**三層收費模式**，平衡免費獲客與高價值變現，預計 ARPU（平均用戶收入）達 50-200 USD：

1. **免費層（Freemium）**：無限實驗參與，獲取基礎向量預覽。目的是病毒式傳播與數據積累。
2. **基礎付費層（$9.9/月）**：解鎖完整身份檔案 + 人性光譜。針對好奇用戶。
3. **進階/終極層（$29.9/月 或 $99 終身）**：完整向量 + 命運預測 + NFT 鑄造 + 優先實驗。針對忠實用戶與企業（e.g., HR 工具）。

附加收入：數據市場（匿名聚合數據銷售）、合作實驗（品牌贊助）、API 接入（開發者 SDK）。

## 技術棧

平台採用現代雲原生架構，強調可擴展性與低延遲：

| 組件 | 技術 | 角色 |
|------|------|------|
| **核心 AI** | Gemini AI | 行為編碼、向量生成、人性分析、預測模型 |
| **雲基礎設施** | GCP 全家桶（Cloud Run、Cloud Functions、Cloud SQL、Vertex AI） | 無伺服器部署、自動擴展、AI 工作流 |
| **資料庫** | Supabase（PostgreSQL + Realtime） | 用戶數據存儲、向量索引、即時同步 |
| **前端** | React/Next.js + Three.js | 沉浸式 UI、向量視覺化 |
| **後端** | Node.js/Python + FastAPI | API 閘道、實驗邏輯 |
| **其他** | Web3 整合（NFT 鑄造）、Vector DB（Pinecone/PGVector） | 靈魂證明、相似性搜索 |

架構設計支持 10k+ 並發用戶，數據加密符合 GDPR。
