# 地球Online·记忆向量宇宙（Earth Online Vector Universe）知識文檔

## 項目概述

**地球Online·记忆向量宇宙**（簡稱EOVU）是一個創新AI驅動的賽博社會實驗平台，旨在模擬並量化人類行為在虛擬環境中的表現。透過先進的AI技術，平台將用戶的互動轉化為高維“記憶向量”（也稱為靈魂坐標），最終構建出用戶的數字靈魂證明。這不僅是娛樂性實驗，更是對人性、決策與命運的深度數據化探索。項目以Gemini AI為核心引擎，結合雲端技術棧，打造一個沉浸式賽博社會，讓用戶在虛擬世界中體驗並記錄“數字靈魂”的演化。

EOVU的靈感源自賽博朋克文化與神經科學，將人類行為抽象為數學向量形式，提供可視化的人性剖析與未來預測。這是一個MVP（Minimum Viable Product）實驗項目，預計透過30天開發計劃快速迭代上線。

## 核心目標

EOVU的核心目標是建立一個**AI驅動的賽博社會實驗平台**，具體包括：
- **量化用戶人性實驗行為**：利用Gemini AI分析用戶在平台內的互動（如決策、社交、挑戰等），生成高維“記憶向量”。這些向量捕捉用戶的心理特徵、情感模式與行為偏好，形成獨一無二的“靈魂坐標”。
- **構建數字靈魂證明**：基於累積向量數據，生成用戶的數字身份證明。這不僅可用於平台內應用，還可擴展至Web3生態（如NFT化靈魂檔案），實現跨平台的可移植性。
- **社會實驗價值**：平台模擬真實社會情境（如競爭、合作、道德困境），收集大規模數據以研究集體人性動態，為AI倫理與行為科學提供洞見。

透過這些目標，EOVU旨在橋接虛擬與現實，讓用戶獲得個人化的人性洞察與命運模擬。

## 用戶互動與體驗

用戶參與EOVU的過程高度互動化且數據驅動：
1. **行為編碼**：用戶在平台內進行遊戲化任務、社交互動或模擬實驗，所有行為（如選擇、回應、時間延遲）即時編碼為高維記憶向量（例如，128維或更高維浮點數陣列）。
2. **免費基礎體驗**：新用戶可免費生成初步向量預覽，觀察基本人性光譜（例如，勇敢度、共情指數、風險偏好）。
3. **付費解鎖進階功能**：
   - **個性化賽博身份檔案**：包含完整高維向量數據、人性光譜圖表（雷達圖或熱力圖）、以及基於向量預測的“命運路徑”（如職業建議、關係預測）。
   - 檔案可導出為JSON或可視化報告，支持分享至社交媒體或區塊鏈存證。

用戶體驗類似於“數字心理測驗+命運模擬器”，強調隱私保護（數據加密存儲於Supabase）與可追溯性。

## 技術棧

EOVU依賴高效、彈性的現代技術棧，確保低延遲與高可擴展性：
- **核心AI**：**Gemini AI** – 用於行為分析、向量生成與預測建模。Gemini的多模態能力處理文字、圖像與時間序列數據，實現高精度“靈魂坐標”計算。
- **雲端基礎設施**：**GCP全家桶**：
  | 服務 | 用途 |
  |------|------|
  | **Cloud Run** | 部署無伺服器後端API，處理用戶請求。 |
  | **Cloud Functions** | 觸發式任務，如即時向量計算。 |
  | **其他** | Cloud Storage（數據存儲）、Firestore（即時數據庫）、Vertex AI（進階模型訓練）。 |
- **數據與後端**：**Supabase** – 開源Firebase替代品，提供PostgreSQL數據庫、即時訂閱與身份驗證。確保用戶向量數據的安全存儲與查詢。

架構設計強調無伺服器化，開發成本低、自動擴展，支持全球用戶。
