# 項目目錄結構知識文檔

## 概述
本知識文檔詳細描述了一個基於 Node.js 的網絡小說項目目錄結構。該項目包含書籍分類存儲、Web 前端頁面以及後端服務器腳本，總計 **9 個目錄** 和 **11 個文件**。項目主要用於管理與展示多種類型的中文網絡小說資源，包括古代言情、恐怖驚悚等熱門類型。

此結構適用於靜態網站托管或簡單的 Node.js 服務器應用，常見於個人博客、電子書庫或內容聚合平台。根目錄作為項目入口，`public/` 提供前端資源，`books/` 存儲分類內容，`server.js` 負責後端服務。

## 部署與使用脈絡
1. **環境要求**：Node.js (v14+)、Python (用於腳本)、npm。
2. **啟動步驟**：
   - `npm install` 安裝依賴。
   - `node server.js` 啟動服務器（默認監聽 3000 端口）。
   - 訪問 `http://localhost:3000` 瀏覽 `index.html`。
3. **維護提示**：
   - 使用 `consolidate_books.py` 更新書籍數據。
   - 檢查 `DELETION_REPORT.md` 追蹤變更。
   - `CATALOG.md` 作為動態生成的目錄索引。
