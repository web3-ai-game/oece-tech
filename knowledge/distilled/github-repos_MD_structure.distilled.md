# 項目結構知識文檔

## 概述
本知識文檔詳細描述一個基於小說分類的靜態網站或文檔存儲項目。項目根目錄包含核心配置文件、腳本和元數據文件，**books/** 子目錄組織了按類型分類的小說資源，**public/** 目錄存放前端入口頁面。總體結構簡潔，支持書籍分類瀏覽和管理。

**總結數據**：
- **目錄數量**：9 個
- **文件數量**：11 個

此結構適用於文學資源庫、個人圖書館或內容聚合網站，可能使用 Node.js 環境（基於 package.json）進行構建和書籍合併處理。

## 根目錄文件
根目錄是項目的核心，包含目錄索引、構建工具和初始化文檔：

| 文件名              | 描述                                                                 |
|---------------------|----------------------------------------------------------------------|
| **CATALOG.md**     | 項目或書籍的總目錄索引，可能列出所有可用小說、分類和導航鏈接。       |
| **DELETION_REPORT.md** | 刪除操作報告，記錄已移除的文件、原因和時間戳，用於審計和版本控制。 |
| **README.md**      | 項目說明文檔，提供安裝、使用指南、貢獻規範和項目背景。             |
| **START.md**       | 啟動指南，可能包含快速開始步驟、依賴安裝和運行命令。                 |
| **consolidate_books.py** | Python 腳本，用於合併或整理 **books/** 目錄中的小說文件（如 TXT、EPUB）。支持批量處理分類數據。 |
| **metadata.json**  | 元數據文件，存儲項目資訊（如版本、書籍總數、作者列表）或書籍索引 JSON。 |
| **package-lock.json** | Node.js 依賴鎖定文件，確保構建環境一致性。                          |
| **package.json**   | Node.js 項目配置文件，定義腳本、依賴（如 Express 或書籍解析庫）和元資訊。 |
| **structure.txt**  | 純文本結構描述，可能為自動生成的文件，記錄目錄樹狀結構。             |

## 子目錄結構

### books/ (書籍分類目錄)
此目錄包含 **6 個子目錄**，按小說類型組織資源。每個子目錄可能存放 TXT、PDF 或 HTML 格式的小說文件，支持分類瀏覽。子目錄內文件數量未指定，但整體貢獻了大部分存儲內容。

| 子目錄     | 描述                                   | 示例內容脈絡                  |
|------------|----------------------------------------|-------------------------------|
| **其他**  | 綜合或未分類小說                       | 現代文學、散文等雜項          |
| **古代言情** | 古典言情小說（如穿越、宮廷題材）      | 《步步驚心》、《甄嬛傳》類   |
| **恐怖驚悚** | 恐怖與驚悚類作品                      | 鬼故事、心理驚悚             |
| **推理懸疑** | 推理偵探與懸疑小說                    | 阿加莎·克里斯蒂、东野圭吾類 |
| **盜墓探險** | 盜墓冒險題材（如摸金校尉傳奇）        | 《��吹燈》、《盜墓筆記》     |
| **網絡小說** | 當代網絡文學（玄幻、都市等）          | 起點中文網流行作品            |

**脈絡補充**：此分類反映華語網絡文學生態，常見於中文小說資源站。**consolidate_books.py** 可能用於從這些子目錄合併生成統一目錄或數據庫。

### public/ (公開靜態資源)
存放前端入口文件，支持 Web 部署（如 GitHub Pages 或 Vercel）。

| 文件名       | 描述                                                                 |
|--------------|----------------------------------------------------------------------|
| **index.html** | 網站首頁入口，包含導航菜單、書籍搜索框和分類鏈接到 **books/** 子目錄。可能整合 CSS/JS 實現響應式設計。 |

## 使用與維護指南
1. **安裝依賴**：`npm install`（基於 package.json）。
2. **運行腳本**：`python consolidate_books.py` 整理書籍。
3. **部署**：將 **public/** 內容上傳至靜態託管服務。
4. **更新**：參考 **CATALOG.md** 和 **DELETION_REPORT.md** 管理內容。
5. **目錄總計驗證**：9 個目錄（根 + books/ 的 6 個 + public/ + 1 隱含）與 11 個文件匹配。

此結構高效、模塊化，便於擴展新分類或整合搜索功能。
