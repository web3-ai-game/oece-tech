# 項目工具與資源知識文檔

## 介紹

本知識文檔是專為項目開發與運作提供的配套工具和資源索引。它彙整了關鍵腳本與導航文件，旨在幫助開發者、製作人員或團隊成員快速定位並應用相關資源。這些工具主要聚焦於視覺化控制（如攝像機運鏡）和工作空間導覽，提升項目效率與協作體驗。

**背景脈絡**：在多媒體項目、虛擬製作或遊戲開發中，攝像機控制腳本和全景工作空間導航是核心組件。前者用於精準鏡頭運動，後者提供項目概覽，確保團隊在複雜環境中高效導航。本索引文件作為「一站式」參考，涵蓋文檔列表、使用指南與實際應用建議。

## 文檔列表

以下是核心配套文檔，按功能分類列出。每份文檔均包含詳細說明與原始碼/配置範例。

### 攝像機與鏡頭控制
- **12-camera-script.md**  
  **描述**：這是一個專用的攝像機腳本，用於鏡頭運鏡控制。它支援多機位切換、平滑軌跡運動（如推拉、搖移、跟蹤）和關鍵幀插值，適用於虛擬攝影棚或即時渲染環境。  
  **關鍵特點**：
  - 支援12個預設攝像機視角。
  - 整合時間線控制與自動化腳本。
  - 相容Unity、Unreal Engine或其他腳本化引擎。

### 工作空間導航
- **21-index-workspace-overview.md**  
  **描述**：工作空間全景導航索引，提供項目全貌導覽。包含互動式地圖、資源樹狀結構和搜尋功能，幫助用戶快速瀏覽模組、資產與設定。  
  **關鍵特點**：
  - 全景（360°）視圖支援，模擬實體工作室巡覽。
  - 動態索引更新，支援版本控制整合（如Git）。

## 使用說明

### 基本使用流程
1. **下載與安裝**：將文檔置於項目根目錄的`docs/tools/`資料夾中。確保依賴環境（如Node.js或Python）已安裝。
2. **存取文檔**：
   - 使用Markdown檢視器（如VS Code、Typora）開啟文件。
   - 對於腳本類文件，直接複製至編輯器執行。
3. **整合項目**：在主工程中引用這些腳本，例如透過`include`或`import`語法載入`12-camera-script.md`的配置。
4. **更新機制**：定期檢查版本標記，透過Git pull同步最新索引。

**注意事項**：
- 所有文件均為純文本格式，便於跨平台使用。
- 如遇編碼問題，確保UTF-8編碼並使用相容編輯器。

## 實際應用建議

### 場景1：虛擬製作流程
- **應用**：在影片製作中使用`12-camera-script.md`控制虛擬攝像機。
- **步驟**：
  1. 載入腳本至時間線軟體（如Adobe After Effects或Nuke）。
  2. 設定關鍵幀：例如，從廣角全景（Camera 1）平滑切換至特寫（Camera 7）。
  3. 搭配`21-index-workspace-overview.md`導航資產庫，快速選取模型。
- **益處**：減少手動調整時間50%，提升鏡頭一致性。
- **範例代碼片段**（來自12-camera-script.md）：
  ```javascript
  // 簡單運鏡範例
  camera.moveToPreset(12, {duration: 3.5, easing: 'easeInOut'});
  ```

### 場景2：團隊協作與項目管理
- **應用**：新成員入隊時，使用`21-index-workspace-overview.md`進行全景導覽。
- **步驟**：
  1. 開啟全景模式，點擊熱點進入子模組（如「資產管理」）。
  2. 結合`12-camera-script.md`測試渲染預覽。
  3. 分享連結至團隊聊天工具（如Slack）。
- **益處**：縮短上手時間，提升跨部門協作效率。
- **最佳實踐**：
  | 角色 | 推薦工具 | 應用提示 |

  | 導演/美術 | 12-camera-script.md | 自訂鏡頭軌跡，預覽故事板 |
  | 工程師 | 21-index-workspace-overview.md | 快速定位腳本與API文件 |
