# 項目資源索引知識文檔

## 概述
本知識文檔作為**項目實用工具與資源的集中索引**，旨在為團隊成員提供快速、便捷的查找入口。透過彙整關鍵工具、文檔與資源，本索引幫助使用者高效定位所需內容，支持項目開發、運維與優化工作。文件聚焦於核心資源清單，並提供擴充建議，以確保長期可用性與導航效率。

**文件目的**：
- 索引項目實用工具與資源，提供快速查找。
- 降低搜尋時間，提升團隊生產力。
- 作為資源管理的活文檔，支持持續迭代。

**適用對象**：項目開發者、設計師、運維人員與相關利益相關者。

**版本資訊**：v1.0（初始發布）  
**最後更新**：2023-10-01（定期審閱機制將確保內容新鮮度）。

### 1. 攝像機腳本文檔
- **描述**：專用於鏡頭運鏡控制的腳本工具集，提供精準的攝像機運動模擬與自動化控制。
- **核心功能**：
  - 支援平移（Pan）、傾斜（Tilt）、變焦（Zoom）等運鏡操作。
  - 整合時間線編輯，適用於虛擬製作、遊戲開發或動畫渲染。
- **使用情境**：虛擬攝影棚、3D 場景導航或實時預覽。
- **訪問方式**：[連結至攝像機腳本文檔](link-to-camera-script-doc)（內部資源，請確認權限）。
- **補充脈絡**：此文檔源自項目早期原型階段，解決手動運鏡效率低下的痛點，已在多個場景中驗證穩定性。

### 2. 工作空間全景導航文檔
- **描述**：提供工作空間的互動式全景概覽工具，支援 360° 視角導航與熱點交互。
- **核心功能**：
  - 全景圖像載入與自適應渲染。
  - 整合導航熱點、資訊疊加與路徑規劃。
- **使用情境**：遠端協作、虛擬辦公室巡覽或資產庫瀏覽。
- **訪問方式**：[連結至工作空間全景導航文檔](link-to-workspace-panorama-doc)（內部資源，請確認權限）。
- **補充脈絡**：設計用於提升團隊空間感知，特別適用於分散式工作環境，已優化多設備相容性（桌面/行動）。
