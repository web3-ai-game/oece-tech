# 項目配套工具與資源索引知識文檔

## 介紹

本知識文檔介紹一個項目配套的工具與資源索引系統，旨在為開發者、製作人員或團隊提供高效的參考與導航支持。這些資源專注於視覺化項目（如影片製作、虛擬實境或遊戲開發）的關鍵組件，包括攝像機控制腳本和工作空間導航工具。

**背景脈絡**：在多鏡頭拍攝或虛擬環境項目中，精準的鏡頭控制和工作空間導航至關重要。傳統手動操作容易導致錯誤或效率低下，而本資源透過腳本化和索引化方式，標準化流程，提高生產力。這些工具適用於Unity、Unreal Engine或其他支援Markdown腳本的開發環境。

**資源目的**：
- 提供實用工具索引，便於快速定位與應用。
- 支援團隊協作，作為項目文檔的核心參考。

## 可用文檔列表

以下是核心文檔列表，每份文檔均為獨立Markdown檔案，包含詳細說明、代碼範例與配置指南。

### 12-camera-script.md
- **描述**：攝像機腳本，用於鏡頭運鏡控制。
- **功能**：
  - 支援12個攝像機的切換、平移、縮放與路徑追蹤。
  - 自動化鏡頭預設（如廣角、特寫、環繞），減少手動調整時間。
- **技術細節**：
  - 基於腳本語言（如C#或Python），整合時間線控制。
  - 相容多平台渲染引擎。

### 21-index-workspace-overview.md
- **描述**：工作空間全景導航。
- **功能**：
  - 提供互動式全景視圖，涵蓋整個工作空間的資源映射。
  - 支援縮放、點擊導航與搜尋功能，快速定位資產、腳本或設定檔。
- **技術細節**：
  - 使用HTML5全景技術或Unity WebGL嵌入，實現360度瀏覽。
  - 包含熱點連結，直接跳轉至相關文檔。

## 使用說明

### 安裝與設定
1. **下載資源**：將文檔置於項目根目錄的`docs/tools/`資料夾。
2. **環境準備**：
   - 安裝Markdown編輯器（如VS Code with Markdown Preview）。
   - 若涉及腳本執行，確保開發環境（如Unity 2021+）已配置。
3. **索引整合**：將`21-index-workspace-overview.md`設為入口頁面，透過瀏覽器或IDE開啟。

### 步驟指南
1. 開啟`21-index-workspace-overview.md`作為起始點，瀏覽全景導航。
2. 導航至`12-camera-script.md`，複製腳本至您的場景控制器。
3. 測試腳本：綁定至空物件，運行鏡頭序列。

**實用提示**：
- 使用版本控制（如Git）追蹤文檔變更。
- 在生產環境中，將腳本打包為Prefab以重複使用。
