# 專案目錄結構知識文檔

## 概述
本專案是一個基於 Node.js 的文檔或圖書管理系統，包含多個根目錄文件、11 個子目錄（其中一個為多層嵌套的圖書分類目錄）、以及支援腳本和元數據文件。總計 **11 個目錄** 和 **14 個文件**。專案結構顯示其可能用於存儲、整理和提供網絡小說、推理懸疑、恐怖驚悚等中文圖書內容，透過 `server.js` 運行本地伺服器，並支援書籍合併處理（`consolidate_books.py`）。

專案根目錄包含文檔說明文件、目錄結構記錄，以及前端靜態資源（`public/index.html`）。目錄設計強調圖書分類，便於管理和瀏覽。

## 子目錄（11 個）
專案包含 11 個子目錄，其中 `books` 為核心目錄，內嵌 5 個圖書分類子目錄，用於組織中文網絡小說和類型文學。其他目錄支援數據結構和前端資源。

### 圖書分類目錄：`books/`
- **位置**：根目錄下。
- **子目錄**（5 個）：專注中文圖書分類，涵蓋言情、懸疑、探險等流行類型。
  
| 子目錄     | 描述 |
|------------|------|
| **其他**  | 雜類圖書，包含不屬於其他分類的內容。 |
| **古代言情** | 古代背景的言情小說，如穿越或宮廷愛情類型。 |
| **恐怖驚悚** | 恐怖、懸疑故事，強調驚悚氛圍。 |
| **推理懸疑** | 偵探、推理和懸疑小說。 |
| **盜墓探險** | 盜墓筆記式探險冒險類型（如《盜墓筆記》風格）。 |
| **網絡小說** | 網絡連載小說，涵蓋玄幻、仙俠等流行網絡文學。 |

### 其他子目錄
| 目錄名稱       | 描述 |
|----------------|------|
| **data/structures** | 數據目錄，用於存儲結構化數據，如目錄樹或 JSON 結構文件。 |
| **public/**   | 公共靜態資源目錄，包含前端入口文件 `index.html`，適用於 Web 伺服器提供頁面。 |

**備註**：`books` 內的 5 個子目錄計入總 11 個目錄中；`public/` 包含至少 1 個文件（`index.html`）。

## 總計統計
- **目錄**：11 個（`books` + 其 5 個子目錄 + `data/structures` + `public/` + 其他 4 個未詳述目錄）。
- **文件**：14 個（根目錄 8 個 + 其他 5 個 + `public/index.html` 計入其他類別）。
- **總大小估計**：依圖書內容而定，`books` 目錄可能佔主要儲存空間。

## 使用建議
1. **啟動專案**：運行 `npm install`（依 `package.json`），然後 `node server.js` 啟動伺服器。
2. **書籍管理**：使用 `consolidate_books.py` 處理圖書檔案。
3. **瀏覽**：透過 `public/index.html` 訪問分類目錄。
4. **開發注意**：檢查 `README.md` 和 `STRUCTURE.txt` 以了解完整結構變更。
