1. **安裝**：`npm install`（基於package.json）。
2. **運行腳本**：`python consolidate_books.py` 更新書籍數據。
3. **啟動服務器**：`node server.js` 或參考`START.md`/`README.md`。
4. **訪問**：瀏覽`http://localhost:3000` 查看`index.html`和書籍分類。
