# 1. 社會工程學基礎

社會工程學 (Social Engineering) 利用人類心理弱點獲取敏感資訊，佔 cyber attack 74% (Verizon DBIR 2024)。防禦重點：意識訓練 + 技術控制。chunk per 攻擊類型，Firebase metadata {attack_type: "phishing"}，Notion gallery 範例 email。

## 1.1 定義與歷史

### 1.11 利用人類心理弱點獲敏感資訊（Kevin Mitnick 'The Art of Deception'）。

Mitnick 自傳：預設信任、權威順從 (Milgram 實驗 65% 服從)。

**心理原則 (Cialdini)：**
- 互惠
- 權威
- 稀缺
- 社會證明

### 1.12 MITRE ATT&CK T1566 Phishing。

T1566.001 Spearphishing，T1566.002 附件。

## 1.2 常見類型

### 1.21 Phishing：假 email，檢查 sender spoof。

**識別：**
- Hover URL (非銀行.com)
- 緊急語言 "帳戶凍結"
- 附件 .exe/.zip

**工具：** VirusTotal scan URL。

### 1.22 Vishing：電話，驗證 callback。

**腳本範例：** "IT部門，您電腦中毒，按1輸入密碼"。防禦：掛斷後官方號碼回電。

### 1.23 Smishing：SMS，勿點不明連結。

**範例：** "包裹延遲，點連結追蹤" → 惡意 APK。

## 1.3 防禦策略

### 1.31 訓練：模擬 phishing KnowBe4。

KnowBe4 平台：模擬攻擊，點擊率降 90%。台灣企業：PhishMe 或本地 SANS。

**年度計劃：**
- Q1: 全員訓練
- Q2: 模擬演練
- Q4: 報告審核

### 1.32 技術：DMARC, SPF, DKIM email驗證。

**DNS 配置：**
```
_dmarc.example.com TXT "v=DMARC1; p=quarantine; rua=mailto:report@"
mail._domainkey TXT selector._domainkey {k=rsa;p=...}
```

## 1.4 個人防護

### 1.41 2FA everywhere，password manager。

推薦：Bitwarden (開源)，YubiKey hardware 2FA。

### 1.42 報告：內部 incident response。

流程：發現 → 截圖 → IT 報告 → 隔離。

**總結**：意識 + 驗證擋 90% 攻擊。向量搜索「phishing 範例」，Notion database 記錄個人事件。
