## 7. 語言學習路線（Python/Java/Go/Rust）
**目標**：多語言精通，依用途選擇。

7.1 **總時長**：每語言35-42小時。背景：語言選擇基於應用，如Python for ML。

7.11 **Python**：Python for Programmers (15h) + Advanced (20h)。

7.12 **Java**：Java Fundamentals (20h) + Multithreading (12h)。

7.13 **Go**：Go for Beginners (15h) + Advanced (18h)。

7.14 **Rust**：Rust Fundamentals (22h) + Systems (20h)。

| 語言 | 基礎課程 | 進階課程 | 總時長 | 應用場景 |
|------|----------|----------|--------|----------|
| Python | Python for Programmers | Advanced Python | 35h | ML/腳本 |
| Java | Java Fundamentals | Multithreading | 32h | 企業後端 |
| Go | Go Programming | Advanced Go | 33h | 雲服務 |
| Rust | Rust Fundamentals | Systems Programming | 42h | 系統效能 |

7.2 **學習順序**：Python → Java → Go → Rust。
