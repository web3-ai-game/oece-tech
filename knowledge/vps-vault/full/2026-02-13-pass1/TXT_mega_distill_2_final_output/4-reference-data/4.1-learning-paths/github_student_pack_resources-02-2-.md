## 2. 資源分類與深度分析

### 2.1 設計資源（🎨 Design Resources）

#### 2.11 Icons8：多媒體資產平台
背景：Icons8 成立於2012年，是專注於數字資產的平台，提供 icons、photos、illustrations 和 music。GitHub 學生包給予3個月免費訂閱，允許無限下載。原理：基於訂閱模式，學生可存取高解析度資產，用於 UI/UX 設計。實例：一名 app 開發學生下載 SVG icons 用於 Flutter 項目，提升介面美觀。

#### 2.12 IconScout：綜合圖標庫
背景：IconScout 擁有490萬+資產，包括 icons、illustrations、3D assets 和 Lottie animations。學生包提供每月60個高級下載，持續1年。原理：配額系統鼓勵定期使用，資產可整合到 Figma 或 Adobe XD。實例：網頁設計師使用 Lottie animations 創建動態載入效果，提高用戶互動。

#### 2.13 Octicons：GitHub 官方圖標
背景：Octicons 是 GitHub 的開源圖標庫，專為 GitHub 介面設計。學生包提供 Figma 文件訪問。原理：開源許可允許 clone 和修改，用於自訂 UI。實例：開發者 clone repo，將 icons 應用於個人 GitHub Pages。

#### 2.14 Visme：視覺化工具
背景：Visme 專注於 presentations、infographics 和 data visualizations。學生包給予3個月 Starter 計劃。原理：模板驅動設計，加速內容創作。實例：學生使用模板製作數據報告，呈現機器學習項目結果。

#### 2.15 Bootstrap Studio：網站設計工具
背景：Bootstrap Studio 是基於 Bootstrap 框架的 GUI 工具。學生包提供學生期間免費授權。原理：拖拽介面生成響應式網站。實例：前端學生設計 portfolio 網站，導出 HTML/CSS 代碼。

### 2.2 教育與教程資源（📚 Educational Resources）

#### 2.21 Educative：互動編程課程
背景：Educative 提供無視頻、互動式課程，涵蓋多種語言。原理：基於實時代碼編輯器，強化動手學習。實例：學生學習 Python 課程，完成內建練習。

#### 2.22 DataCamp：數據科學平台
背景：DataCamp 專注於 data science 和 analytics。原理：項目導向學習，結合 R 和 Python。實例：分析真實數據集，如 Kaggle 競賽。

#### 2.23 FrontendMasters：前端開發課程
背景：FrontendMasters 提供專家級視頻課程。原理：深度講座加代碼挑戰。實例：學習 React 課程，構建單頁應用。

#### 2.24 Scrimba：互動教程
背景：Scrimba 使用可暫停代碼錄製。原理：模擬真實編程環境。實例：JavaScript 教程中編輯講師代碼。

#### 2.25 Codedex：編程學習
背景：Codedex 提供初學者友好課程。原理：遊戲化學習。實例：解決編程 puzzle。

#### 2.26 GoRails：Ruby on Rails 教程
背景：GoRails 專注 Rails 框架。原理：視頻加代碼。實例：構建 CRUD 應用。

#### 2.27 SymfonyCasts：PHP 教程
背景：SymfonyCasts 針對 Symfony 框架。原理：實戰項目。實例：開發 API 後端。
